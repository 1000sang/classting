## Requirement
- docker
- docker-compose

## Installation
```angular2html
# docker 설치
$ brew install cask docker

# docker-compose 설치
$ brew install cask docker-compose

```

## Running the app

```
$ docker compose up
```

## Test
```angular2html
$ cd server

# test coverage
$ npm run test:cov
```
